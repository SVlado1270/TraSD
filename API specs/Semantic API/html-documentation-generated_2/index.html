Semantic API

A Flask API which is a wrapper for the SPARQL queries. We defined multiple relations in our ontology between traffic signs. counterPart relation means that a sign is the counterpart for another like Speed Limit 80 and End Of Speed Limit 80. isOpposedTo relation means that the meaning of a sign is the opposite of another.
